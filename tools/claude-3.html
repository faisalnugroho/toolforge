The Claude family most people still search for

Claude 3 was the model family that broke OpenAI's lead in early 2024. Three tiers — Opus (smartest), Sonnet (workhorse), Haiku (fastest) — each tuned for different jobs. By 2026 Claude 4 has replaced it on the consumer chat product (claude.ai), but the Claude 3 family is still alive and well on the Anthropic API, often at a discount. Many production apps you use daily are still running Sonnet 3.5 or 3.7 under the hood.

Who it's for: Developers building production apps on a budget, anyone who has heard "Claude 3" in podcast ads and wants to understand what it actually was, enterprises still on Claude 3 inference contracts.

The Claude 3 tiers

Opus The original "smartest"

Claude 3 Opus was the first Claude that consistently beat GPT-4 on graduate-level reasoning, coding contests, and long-form analysis. The model that made people switch from ChatGPT in 2024.

Sonnet The workhorse

Claude 3.5 Sonnet became the default Claude model — fast enough for daily use, smart enough for production. The model that powered Cursor, Notion AI, and hundreds of startups in 2024-2025.

Haiku The speedster

Claude 3 Haiku was Anthropic's "fast and cheap" tier — used for classification, routing, and high-volume lightweight tasks. Still useful in 2026 for any task that doesn't need deep reasoning.

200K context Big memory (for 2024)

Claude 3 introduced 200K-token context windows — massive for 2024, surpassed by Claude 4's 1M tokens in 2025. Still plenty for most code and document workloads.

The honest take

✓ What works

  • Still available via API — perfect for budget-conscious production use
  • Well-documented, well-understood behavior (less surprises than newer models)
  • Sonnet 3.5 is widely regarded as the best code-reviewer per dollar
  • Cheaper than Claude 4 — about 1/3 the price per token
  • Vision support (image input) was a Claude 3 first

✗ What doesn't

  • Not available on the consumer claude.ai chat in 2026 — replaced by Claude 4
  • Slower than Claude 4 Sonnet on equivalent tasks
  • Smaller context window (200K vs Claude 4's 1M)
  • Newer features (Projects, Artifacts) are Claude 4-only
  • Anthropic will eventually deprecate — migrate before EOL

Verdict

If you are a developer building production AI features on a budget, Claude 3 Sonnet via the Anthropic API is still an excellent choice in 2026 — most of the benchmarks it led in 2024 are still within 5-10% of Claude 4 Sonnet, at one-third the price. If you are a consumer, just use Claude 4 directly. The Claude 3 era ended for end users, but for API builders it remains a high-value option.
