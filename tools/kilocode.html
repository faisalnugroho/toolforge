An autonomous coding agent that lives in your editor

Kilo Code is an open-source AI agent that installs as a VS Code extension and plans, writes, and debugs across your whole codebase. Point it at a task and it edits multiple files, runs terminal commands, and iterates — no need to switch to a proprietary AI IDE.

Who it's for: Developers who want agentic coding inside the VS Code they already use, without paying for a separate editor.

Key features

🤖 Agentic edits

Give it a task; it plans, edits multiple files, and runs terminal commands.

🔧 VS Code native

Installs as an extension — keep your themes, keybindings, and extensions.

🐛 Debug & refactor

Finds bugs, explains errors, and proposes fixes with full context.

🔓 Open source

Apache-2.0 licensed; bring your own model (Claude, GPT, or local).

The honest take

✓ What works

  • Free and open source
  • Works inside VS Code you already use
  • Supports many models (Claude, GPT, local)
  • Active community development

✗ What doesn't

  • Less polished than Cursor's Composer
  • Agent can go off-track on big tasks
  • Setup varies by model provider
  • Smaller ecosystem than proprietary IDEs

Verdict

The best free, open entry into agentic coding if you don't want to leave VS Code. For a fully integrated AI IDE, Cursor or Windsurf still feel tighter — but Kilo Code is closing fast.
