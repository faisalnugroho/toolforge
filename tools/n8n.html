The fair-code automation platform for technical teams

n8n (pronounced "n-eight-n") is the workflow automation tool that finally gives engineers what they actually want: full source code access, self-hosting on your own infrastructure, and a visual editor that doesn't hide the underlying logic. With 400+ pre-built integrations and native AI nodes for OpenAI, Anthropic, Hugging Face, and any LLM via HTTP, n8n is the platform of choice for teams that need automation without vendor lock-in.

Who it's for: Engineering teams, technical operations folks, indie hackers, and anyone who wants the power of Zapier/Make without the per-task pricing or closed-source constraints.

Key features

AI nodes Native LLM integration

First-class nodes for OpenAI, Anthropic Claude, Google Gemini, Mistral, and Hugging Face. Chain prompts, parse structured outputs, and build AI agents with tool-calling — all visually.

400+ nodes Massive integration library

Pre-built nodes for Slack, Notion, Airtable, HubSpot, Stripe, GitHub, Postgres, Google Sheets, Salesforce, and hundreds more. Anything not built-in can be hit with the HTTP node.

Self-host Fair-code license

Run n8n on your own infrastructure (Docker, Kubernetes, even a Raspberry Pi) with the Sustainable Use License. No per-execution fees, no data leaving your network. Cloud option available if you don't want to manage servers.

Code nodes JavaScript & Python

Drop into actual code (JS or Python) anywhere in your workflow. Write custom transformations, call internal APIs, do calculations. Visual flow + real code = best of both worlds.

The honest take

✓ What works

  • Open-source and self-hostable — no vendor lock-in, no per-task pricing
  • Native AI nodes (OpenAI, Claude, etc.) make building AI agents fast
  • Code nodes let you escape the no-code limits when you need to
  • Active community, 400+ integrations, regular releases
  • Free for self-hosters (fair-code license)
  • Webhooks, scheduling, error handling, retries all built in

✗ What doesn't

  • Steeper learning curve than Zapier or Make
  • Cloud Starter plan at $24/mo is limited to 2,500 executions — Pro is $60/mo
  • Self-hosting means you handle updates, backups, scaling
  • UI is dense and can feel overwhelming for non-engineers
  • Some integrations are community-built and less polished than Zapier's

Verdict

If you're an engineer or technical ops person who has ever cursed at Zapier's per-task pricing or Make's hidden execution costs, n8n is the answer. Self-host it for free, build AI agents in an afternoon, and own your data. For pure no-code users, Make is friendlier. For SMBs who want hands-off, Zapier is simpler. For everyone technical, n8n wins.
