The community's answer to Cursor

PearAI is a fork of VSCode that ships with AI features out of the box — but unlike Cursor, Windsurf, or Trae, you bring your own API key (BYOK) and pick the model. It started as a fork of Continue.dev and has since grown into a serious editor in its own right, with 80K+ GitHub stars and active monthly releases.

Who it's for: Developers who want Cursor-class UX but refuse vendor lock-in. Open-source purists, privacy-conscious teams, and anyone already paying for OpenAI/Anthropic API credits.

Key features

BYOK Bring your own key

Plug in OpenAI, Anthropic, Google, Mistral, Groq, OpenRouter, or any OpenAI-compatible endpoint. PearAI doesn't take a cut — you pay the model provider directly.

Cmd-I Inline AI commands

Highlight code, describe the change. PearAI rewrites the selection using your chosen model. Same flow as Cursor and Trae.

Sequential Multi-step agent

PearAI's Sequential agent reads multiple files, plans the change, and applies edits across your project. Not as autonomous as Cursor Composer, but more transparent.

OSS 100% open source

Apache 2.0 licensed. Audit the code, self-host the AI server, fork it for your team. No telemetry without your consent.

The honest take

✓ What works

  • Truly open source — Apache 2.0, not source-available
  • BYOK means no surprise bills; pay model providers directly
  • Works with any model — even local Ollama or LM Studio
  • Active Discord community, fast PR turnaround
  • Privacy mode is the default, not opt-in

✗ What doesn't

  • Sequential agent lags Cursor Composer on multi-file tasks
  • Smaller extension marketplace than vanilla VSCode
  • You manage your own API costs — no bundled free tier
  • Some advanced features (background agents, codebase embeddings) still in progress
  • Smaller team means slower shipping than Cursor/Trae

Verdict

PearAI is what you reach for when you want Cursor's UX without giving a venture-backed startup access to your codebase. If you're already paying for OpenAI or Anthropic API credits, PearAI is genuinely free forever. The Sequential agent isn't as polished as Composer yet, but the open-source moat is real.
