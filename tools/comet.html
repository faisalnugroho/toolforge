A browser that does your web research for you

Comet is Perplexity's Chromium-based browser with a built-in AI assistant that lives in the sidebar. Highlight any text, ask a question, get a cited answer without leaving the page. It's the first browser that feels designed for the post-search era — you don't visit websites, you ask questions, and the agent fetches, reads, and summarizes the answer right there.

Who it's for: Knowledge workers, researchers, shoppers, and anyone who spends more than an hour a day in a browser and is tired of opening 12 tabs to answer one question.

Key features

Address bar AI-powered address bar

Type a question, get a cited answer. No need to navigate to Google first. Comet reads the top results, synthesizes, and shows inline sources.

Cursor Highlight-to-ask

Highlight any text on any page and ask the assistant anything — explain, summarize, translate, rewrite, or pull a quote. Works in every site including Gmail, Notion, GitHub, and PDF docs.

Agent mode Multi-step web research

Comet can chain actions: open 5 tabs, compare products across sites, fill forms, book appointments, and even checkout. Not perfect, but better than any other browser assistant.

Memory Perplexity Spaces

Pin research, snippets, and pages to Spaces — Perplexity's AI knowledge base. Comet integrates so you can search your saved research the same way you search the web.

The honest take

✓ What works

  • First browser that genuinely makes 'just Google it' obsolete
  • Highlight-to-ask works on literally every page — huge productivity boost
  • Comet Assistant can fill forms, click buttons, and complete multi-step flows
  • Free for Perplexity Pro subscribers (already a great deal)
  • Tab manager and Spaces integration is genuinely useful for ongoing projects

✗ What doesn't

  • Slower than Chrome for raw page loads (Chromium fork overhead)
  • Agent mode is still hit-or-miss on complex multi-step tasks
  • Limited extension library — many Chrome extensions don't install yet
  • Some sites detect and block the AI assistant (Twitter, banking)
  • Mobile is still a closed beta — iOS/Android not generally available

Verdict

Comet is the first browser that I'd actually recommend to a non-technical friend. If you're already paying for Perplexity Pro ($20/mo), Comet is a no-brainer add-on. Even if you're not, the free tier is generous enough to replace Chrome for research-heavy work. The agent mode isn't quite there yet, but the highlight-to-ask feature alone will save you 30+ minutes a day.
