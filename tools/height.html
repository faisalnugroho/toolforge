Project management that gets out of your way

Height is the project tool that feels like a chat app. You create tasks in natural language ('ship the onboarding fix by Friday'), and AI assigns, prioritizes, and links them. Built for engineers who hate project management but need it anyway.

Who it's for: Engineering and product teams who want lightweight, keyboard-first planning.

Key features

Natural language Type to create

Write a sentence; Height parses assignee, due date, and labels automatically.

AI sub-tasks Auto-breakdown

Ask it to break a big initiative into shippable tasks with owners.

Smart lists Views that think

Dynamic filters that update as work changes; AI suggests the right view.

Integrations GitHub + Linear-style

Two-way sync with GitHub, Slack, and Figma so tasks track real progress.

The honest take

✓ What works

  • Fast, keyboard-first UX engineers actually like
  • AI task creation genuinely saves time
  • Clean, modern interface
  • Free tier is usable
  • Great GitHub integration

✗ What doesn't

  • Less suited to non-engineering teams
  • Reporting is lighter than enterprise PM tools
  • Smaller ecosystem than Asana/Monday

Verdict

For dev teams that find Jira heavy and Notion too loose, Height hits a sweet spot. The AI task parsing alone is worth the $9/user/mo — it turns a 3-minute ticket into a 10-second sentence.
