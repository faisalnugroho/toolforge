The fastest way to get more done with Metaphor (Exa)

Metaphor (now Exa) is a search engine built on language models. Instead of matching keywords, it predicts the links an AI would cite for your query — so "find me ML papers on diffusion models" returns the actual papers, not SEO pages. Hugely popular with researchers and builders.

Who it's for: Researchers, developers, and recruiters who need high-signal link retrieval, not ads.

Key features

Neural Link prediction

Returns the URLs an LLM would reference — high precision.

API Programmatic

Query millions of pages via a clean REST/Python API.

Find similar Related links

Drop a URL; get the most similar resources.

Autoprompt NL to query

Describe what you want; it writes the search for you.

The honest take

Metaphor (Exa) earns its place in a modern AI stack for anyone who values search workflows done right. For Free / $5/mo it punches above its weight — and the free tier is genuinely usable. Worth a look before you commit to a heavier alternative.
