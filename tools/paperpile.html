Reference management for the Google Docs generation

Paperpile is a browser-based reference manager built around Google Docs and Google Drive. Import references with one click from PubMed, arXiv, or JSTOR, organize them in Drive, and cite inline in Google Docs with a clean plugin. It is the natural choice for teams already living in the Google ecosystem.

Who it's for: Researchers and students who write in Google Docs and want citations that "just work" without installing desktop software.

Key features

GDocs Google Docs integration

Insert citations and build bibliographies natively inside Google Docs.

Browser Browser-based

Everything runs in Chrome — no heavy desktop app, accessible from any machine.

Import One-click import

Pull references from PubMed, arXiv, JSTOR, and 1,000+ other sources instantly.

Mobile Mobile app

Browse and search your library, and save papers from your phone.

The honest take

✓ What works

  • Best-in-class Google Docs citation experience
  • Zero-install, fully browser-based
  • Clean Drive-based organization
  • Great for collaborative Docs writing

✗ What doesn't

  • Google-centric — weaker for Word users
  • Subscription only, no lifetime license
  • Limited offline capability
  • Smaller community than Zotero/Mendeley

Verdict

If your writing life is in Google Docs, Paperpile is the smoothest reference manager you can buy — $9/mo is cheap for the time it saves. Word-heavy users should look at EndNote or Mendeley instead. Start with the trial before committing.
