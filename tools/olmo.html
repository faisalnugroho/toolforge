The reproducible model

AI2's OLMo is the rare model released with everything: weights, training data, code, and logs. OLMo 2 reaches 32B-class quality while letting anyone reproduce or extend the work from scratch.

Who it's for: Researchers, academics, and orgs that require fully reproducible, auditable, self-hosted models.

Key features

Open everything Data + code

OLMo publishes training data, code, and logs — not just weights — for true reproducibility.

OLMo 2 32B class

The second generation rivals much larger closed models on standard benchmarks.

Research-first Science friendly

Built so researchers can study, ablate, and extend every training decision.

Self-host No vendor

Run it on your own hardware with standard open tooling.

The honest take

✓ What works

  • Genuinely open — data, code, and all
  • Great for research and audit
  • Apache-2.0, safe to build on
  • Strong quality for an open model
  • No API dependency

✗ What doesn't

  • You run the infra yourself
  • Not as polished as hosted rivals
  • Fewer tuned variants than Llama
  • Benchmark gaps vs frontier

Verdict

OLMo is the gold standard for teams that must audit and reproduce every part of a model. If you just want an API, a hosted model is easier — but for science and sovereignty, nothing is more open.
