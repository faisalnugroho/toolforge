The honest take

OpenAI's fast multimodal flagship -- still the all-rounder for most people.

Who it is for: Anyone who wants a fast, capable AI that handles text, image, voice, and code in one place. The default "just works" assistant for 200M+ weekly users.

Key features

Speed 320ms response

The fastest multimodal model in production. Voice conversations feel like talking to a human.

Voice Real-time voice

Hold a conversation with interrupt, emotion, and tone. Five voices, each with distinct personality.

Vision Native vision

Upload photos, screenshots, diagrams. GPT-4o "sees" them like a person, not an OCR pipeline.

Canvas Canvas mode

Side-by-side editing for writing and code. The model suggests inline edits you accept or reject.

Search Web + memory

Searches the live web, remembers preferences across sessions, cites sources inline.

The honest take

✓ What works

  • Cheapest capable multimodal model ($20/mo for the lot)
  • Best voice UX on the market -- interruptions feel natural
  • Image generation via DALL-E 3 built in (still solid)
  • GPTs store lets you install niche assistants without coding
  • Mobile apps are excellent

✗ What does not

  • Long-context reasoning weaker than Claude 4.5 (128K vs 500K)
  • Sycophantic -- tends to agree even when wrong
  • Rate limits on Plus feel stingy for power users
  • Reasoning tier (o3-pro) costs 10x more

Verdict

GPT-4o remains the best "default assistant" for general use. For specialized work -- long docs, code-heavy reasoning, agentic tasks -- Claude 4.5 or Gemini 3 Pro are better. For everyone else, GPT-4o is still king.
