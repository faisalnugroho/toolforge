The codebase-aware AI assistant

Sourcegraph Cody is the AI coding assistant that actually understands your entire codebase. Unlike Copilot or Cursor that work on individual files, Cody indexes your whole repo and answers questions like 'where do we handle authentication?' with exact file references. For developers working on large codebases, Cody is a force multiplier.

Who it's for: Developers working on large codebases, teams, anyone who needs codebase-wide context.

Key features

🔍 Codebase indexing

Indexes your entire repo. Asks questions about the whole codebase, not just one file.

🧠 Context-aware

Knows your conventions, libraries, patterns. Suggests code that fits your style.

🔌 IDE integration

VSCode, JetBrains, Visual Studio, Neovim, Emacs. Works where you work.

📚 Documentation search

Indexes your docs too. Asks questions about both code and documentation.
