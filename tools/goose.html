The short take

Honest, hands-on review of Goose in 2026. Block’s open-source AI engineering agent that ships full features autonomously. Free, runs locally, MIT-licensed.

Who it's for: Block’s open-source AI engineer that ships features end-to-end — free, local, MIT-licensed.

Key features

🚀 Auto Truly autonomous engineering

Give Goose a ticket (“add OAuth2 login to /api/users”) and it plans, writes code, runs tests, and opens a PR. No per-step approval needed — runs until done or stuck.

💻 Loca Local-first, BYO model

Runs on your machine. Bring your own model (Claude, GPT, Gemini, local Ollama). Your code never leaves your laptop unless you choose. MIT-licensed.

🔧 15+ 15+ tool integrations

Built-in shell, file editor, web browser, git, GitHub, Postgres, and 10+ more. Extensions let you add internal tools (Datadog, Linear, Sentry).

🧠 Memo Session memory + planning

Goose keeps a planning todo list and remembers context across long-running tasks. Can resume a session that crashed or ran out of context.

🏦 Bloc From Block (Square, Cash App)

Built by the team behind Cash App and Square. Battle-tested in production at a $50B fintech. Shipped to the world as open-source in early 2026.

The honest take

✓ What works

  • Truly free — no seat fees, no usage caps, MIT-licensed
  • Local execution means your code stays on your machine
  • Autonomous end-to-end execution beats Aider/Cline for greenfield work
  • Model-agnostic — swap Claude for GPT-5 or local Llama 4
  • Backed by Block — not going away

✗ What doesn't

  • Still young (v0.x) — rough edges, occasional hangs on complex tasks
  • No hosted/SaaS option — you set up the runtime yourself
  • Planning loops can spin on ambiguous tickets
  • Smaller community than Cursor, Windsurf, Cline (10K+ Discord members)

Verdict

Goose is the most interesting open-source AI engineer of 2026. If you’re already running Cursor or Claude Code, Goose won’t replace them today — but it’s the right pick when you need autonomous execution, local-first privacy, or zero per-seat cost. Watch this space; the v1.0 release will be a moment.
