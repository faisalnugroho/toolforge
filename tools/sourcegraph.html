Find any line in any repo, instantly

Sourcegraph made code search fast across thousands of repositories. Its companion, Cody, is an AI coding assistant that grounds answers in your actual codebase — not just the open file — so refactors and explanations stay accurate at monorepo scale.

Who it's for: Engineering orgs with large or multi-repo codebases who need search and AI help that understands the whole system.

Key features

Search Cross-repo code search

Regex and structural search across every repo your team owns, with saved searches and alerts.

Cody AI coding assistant

Chat and autocomplete grounded in your codebase, with context from linked repos.

Context Accurate answers

Cody cites files and symbols so you can verify every suggestion.

Integrations GitHub, GitLab, Bitbucket

Works where your code lives; self-host for maximum privacy.

The honest take

✓ What works

  • Best-in-class code search, bar none
  • Cody understands large monorepos
  • Self-host option keeps code private
  • Generous free tier

✗ What doesn't

  • Cody trails Cursor/Claude on agentic edits
  • Self-hosting needs ops effort
  • UI has a learning curve
  • Enterprise pricing is custom

Verdict

For big codebases, Sourcegraph search is unbeatable and Cody is a solid everyday copilot. The free self-hosted tier is enough for most teams to start.
