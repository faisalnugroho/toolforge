Design that devs actually want to inherit

Penpot is the leading open-source design platform. Unlike Figma, its layout engine is based on CSS flexbox and grid, so what designers build maps closely to what engineers ship. You can self-host it for free, which is why privacy-conscious teams and open-source projects gravitate to it.

Who it's for: Designers and frontend teams who want Figma-like collaboration without vendor lock-in or per-seat fees.

Key features

Open source

Self-host or use the free cloud; no lock-in, auditable code.

Flex & grid layouts

Native CSS flexbox/grid so designs translate cleanly to code.

Prototyping

Interactive flows, transitions, and comments for handoff.

Dev mode

Inspect, copy CSS, and export tokens developers can use directly.

The honest take

✓ What works

  • Free and self-hostable — no seat tax
  • Layout model maps to real CSS
  • Strong collaboration features
  • Active open-source community

✗ What doesn't

  • Smaller plugin ecosystem than Figma
  • Performance lags on very large files
  • Onboarding differs from Figma muscle memory
  • No offline desktop app yet

Verdict

Penpot is the most credible Figma alternative for teams that care about ownership and code parity. Use it when you want design tooling that your engineers actually respect.
