The dev tool Cursor users keep coming back to

Aider is an open-source AI pair programmer that lives in your terminal. Unlike IDE-locked tools like Cursor, Aider works with any editor, runs on any model (Claude, GPT, local LLMs), and understands your git history. Multi-file refactors that take 20 minutes in Cursor take 5 in Aider.

Who it's for: Developers who prefer the terminal, want git-native AI, or need to use local/self-hosted models.

Key features

💻 Terminal-native

Works in any terminal, any editor. Vim users love it.

🔀 Git-aware

Auto-commits after each change. Review the diff in git like normal code.

🧠 Multi-model support

Claude, GPT-4, DeepSeek, local Ollama. Switch models per task.

📁 Multi-file edits

Refactor across 50 files with one prompt. Maintains context across the codebase.

The honest take

✓ What works

  • Free and open-source
  • Works with any model (incl. local)
  • Git integration is unmatched
  • Lightweight — no IDE lock-in

✗ What doesn't

  • Terminal-only (no GUI)
  • Setup requires API key configuration
  • Less hand-holding than Cursor for beginners
  • Steeper learning curve
