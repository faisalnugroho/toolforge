How to use AI at every stage of game dev

Concept & prototyping

  • Concept art: Midjourney generates 100 mood-board variants in 30 minutes. Iterate quickly. The art is the pitch.
  • Character designs: Prompt for 30-50 character variants in your style, then refine the best 5-10.
  • UI/UX mockups: Midjourney or Krea for initial screen layouts, then build the real UI in code.
  • Marketing art: Steam capsule, key art, social media — all from your concept art workflow.

Code

  • Game scripts: Cursor + Claude/GPT-4o for player controllers, enemy AI, inventory systems. Compose multi-file features in minutes.
  • Shaders: "Write me a water shader with caustics and foam" — Claude produces clean Shader Graph code. You tweak the look.
  • Tooling: Editor extensions, build scripts, asset importers. Boring but AI handles it perfectly.
  • Bug fixing: Paste the error, the relevant code, and "what I've tried." Cursor fixes it 80% of the time.

Art & 3D

  • Sprites and 2D art: Krea for real-time generation. Midjourney for hero sprites you can upscale and re-color.
  • Tileable textures: Krea or Midjourney with "--tile" flag. Generate ground, walls, foliage in batches.
  • 3D models: Meshy and Tripo3D for character and prop generation. Output FBX, retopologize, rig in Blender.
  • Animation reference: Use video generation models (Kling, Runway) to generate reference animations.

Audio

  • Soundtrack: Suno generates 30+ minutes of cohesive soundtrack from a genre + mood prompt.
  • Sound effects: ElevenLabs Sound Effects generates custom SFX from text. "Sword hitting wooden shield with metallic ring."
  • Voice acting: ElevenLabs voices for every NPC. Clone a real voice actor (with permission) for the protagonist.
  • Adaptive music: Suno stems can be layered for dynamic music systems.

The verdict

For 2D indie games (pixel art, side-scrollers, visual novels): Cursor (code) + Krea (sprites) + Suno (music) + ElevenLabs (voice). Total: ~$60/mo.

For 3D indie games (Unity, Unreal, Godot): Cursor (code) + Midjourney (concept) + Meshy or Tripo3D (3D assets) + Suno (music) + ElevenLabs (voice). Total: ~$100/mo.

For mobile/casual: Skip 3D generation (too much cleanup). Use Cursor + Krea + Suno. Total: ~$50/mo.

Watch-outs

  • Steam disclosure: Valve requires you to disclose if your game uses "live" generative AI (creating content at runtime). Pre-generated assets baked into the build don't need disclosure.
  • Voice cloning: Don't clone voices without written permission. SAG-AFTRA rules still apply even for indie devs using AI voices commercially.
  • Asset cleanup: AI-generated 3D models need retopology, UV unwrapping, and rigging before they ship. Plan 2-3x the cleanup time vs generation time.
  • Style consistency: AI tools drift. Lock in your style with reference images, seeds, and consistent prompts across thousands of asset generations.
