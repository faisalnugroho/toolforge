DeepSeek released two flagship open-weight models in late 2025/early 2026. R1 is a reasoning-first model that thinks step-by-step and rivals OpenAI o1 on math. The verdict

Use DeepSeek R1 for hard reasoning, math, science, and coding challenges — anywhere you'd reach for OpenAI o1. Use DeepSeek V3 for general chat, writing, translation, and any task where latency matters. R1 is "the slow thinker"; V3 is "the fast generalist." Many self-hosters run both and route by task.

Winner: Tie — pick R1 for reasoning, V3 for general use

How we tested

We ran both models locally via vLLM on an 8xH100 node, and via the DeepSeek API (R1: $0.55/M tokens, V3: $0.27/M tokens) for hosted benchmarks. We tested 60+ tasks: math (AIME), code (HumanEval), multilingual translation, long-form writing, and agentic tool use. We don't accept payment for reviews.
